# DataStructuresAssignments2026

## Description
This project includes programming assignments from Metropolitan State University's Data Structures course (ICS 240) that I have completed in the Spring 2026 semester (January - May 2026). Each Assignment directory is a separate Java project that was written in the Eclipse IDE. Detailed descriptions of each assignment's goals are in the .docx files. For Assignments 1-3, a text file name will be requested via the command line; an example text file, computers.txt, can be used. 

### Assignment 1
In this assignment, I practiced writing Java code using files, Objects, and arrays. 

### Assignment 2
In this assignment, I implemented a sorted collection called ComputerSortedArraySet where 
    1. Computer is the class of interest implemented starting in Assignment 1,
    2. the collection implements the set semantics (no duplicates allowed), and
    3. the Computer instances contained in the set are maintained in sorted order.

### Assignment 3
In this assignment, I implemented an Ordered List. The collection stores Computer class instances in a linked list in sorted order. From an unsorted input file, the data for the Computer instances are read in and added to the Ordered List. I implemented methods for inserting, removing, and reading items from specific positions in the linked list, without using the LinkedList class from the Java library. I further maintained multiple Ordered Lists that stored subclass objects of the Computer class. 

### Assignment 4
In this assignment, I practiced using the Queue and Stack data stuctures. I implemented various static methods which operated on the Queue and Stack data structures provided in the Java Collections Framework.
